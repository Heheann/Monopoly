# 景點圖片放置說明

請把你要顯示在彈窗中的景點圖片放在此資料夾，檔名請對應 `board.json` 的 `tile.id`，副檔名使用 `.jpg`。

## 必備檔名

- `pier2.jpg`
- `kmc.jpg`
- `british_consulate.jpg`
- `cijin_fort.jpg`
- `chengcing_lake.jpg`
- `fo_guang_shan.jpg`
- `lotus_pond.jpg`
- `rainbow_church.jpg`
- `tower85.jpg`
- `love_river.jpg`
- `liuhe_night_market.jpg`
- `ruifeng_night_market.jpg`

## 尺寸建議

- 建議比例：`4:3`
- 建議尺寸：`1200 x 900` 以上

如果某張圖尚未放入，遊戲會自動顯示預設 placeholder，不會影響流程。
